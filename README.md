# Time
Bot Do discord de bate ponto com salário automático para staffs baseado em coins (pode ser usado apenas como bate ponto comum);


This is the Discord Bot to count time, as companies do.

This bot automatically pays your staffs using coins.

Dependencies:

`npm install discord.js sqlite3 axios express dotenv luxon big.js && npm install --save-dev nodemon`
